# 🏠 Indian House Price Prediction

## 📌 Project Summary
This project predicts house prices for Indian cities based on features like location, area, BHK, bathrooms, parking, and furnishing.

## 📂 Dataset
- `indian_house_prices.csv` (50 rows, realistic values for Delhi, Mumbai, Bangalore, etc.)

## ⚙️ ML Models
- Linear Regression
- Random Forest Regressor

## 📈 Metrics
- R² Score
- RMSE
- Actual vs Predicted plot

## 🚀 How to Run
1. Clone this repo.
2. Install libraries: `pip install pandas numpy matplotlib seaborn scikit-learn joblib`
3. Run `indian_house_price_prediction.ipynb` in VS Code Jupyter.
4. Check predictions & plots.
